About Our Project

The Automated Plant Caretaker, created by Emmet Blassingame and Blake Brown, is a smart gardening system that ensures the best care for your plants. Using various sensors and a Raspberry Pi, it automates watering and monitors plant health.

Core Features

Automated Watering

Adjusts watering schedules based on real-time soil moisture and weather data.

Environmental Monitoring

Monitors ambient conditions like light, temperature, and humidity for optimal growth.

Remote Control & Monitoring

Allows users to monitor and control their garden remotely through a web application.
